# AI Phishing Email Detector

A cybersecurity tool that analyzes email content and detects phishing indicators using rule-based risk scoring.

## Features
- Urgency detection
- Financial bait detection
- Suspicious link analysis
- Sender domain spoof detection
- Risk scoring (0–100)
- Security explanation output

## How to Run
python detector.py
